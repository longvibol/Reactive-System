    @Override
    public Mono<OwnerResponse> register(OwnerRegisterRequest request) {
    	log.info("Owner registration requested");

        OwnerRegisterRequest normalized = normalizer.normalize(request);

        Owner draft = ownerMapper.toOwnerDraft(normalized);
        Owner pending = ownerFactory.newPendingOwner(draft);

        return registrationValidator.validate(normalized)
            .then(ownerRepository.save(pending))
            //.doOnSuccess(saved -> log.info("Owner registered successfully. ownerId={}", saved.getId()))
            .map(ownerMapper::toResponse);
    }

- create new object for pendding status : newPendingOwner.class

    public Owner newPendingOwner(Owner draft) {

        Instant now = Instant.now(clock);

        return Owner.builder()
            .id(draft.getId()) // usually null; DB generates the ID
            .email(draft.getEmail())
            .phone(draft.getPhone())
            .status(OwnerStatus.PENDING) // enforce lifecycle rule
            .createdAt(now)
            .updatedAt(now)
            .build();
    }


.status(OwnerStatus.PENDING) // enforce lifecycle rule ដែលយើងអោយវា បង្កើត User នឹងទៅជា Pendding សិន

What StringUtils.hasText use for :

- Not Null: The object exists in memory.

- Not Empty: The length is greater than 0 ("").

- Not Just Whitespace: It contains at least one non-whitespace character (it ignores spaces, tabs, and newlines).
